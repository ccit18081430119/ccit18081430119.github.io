Hexo本地搭建

四月 05, 2020

1.准备工作

安装 Node.js (到官网安装适合自己系统的版本即可)

安装Git (到官网安装适合自己系统的版本即可)

2.安装Hexo

前面git和nodejs安装好后，就可以安装hexo了，你可以先创建一个文件夹blog，然后cd到这个文件夹下（或者在这个文件夹下直接右键git bash打开）。

在CMD窗口使用下面的命令

npm install hexo-cli -g

3.搭建本地博客

Hexo根目录创建

在cmd输入如下指令

md blog

初始化Hexo博客

在cmd输入如下指令（首先cmd需进入blog文件夹，进入指令：cd blog)

hexo init

4.本地预览

在cmd输入如下指令

hexo s
